Using Transformer to forecast pressure

Importance: With LLM, internal combustion engine would be able to forecast the working condition in the future. This enables control mechanisms (timing mechanisms, fuel injection, lightning etc) to dynamically trigger based on situation. Has strong potential to further increase thermal efficiency. 

Dataset: Mukhsin, Muammar (2021), “Diesel-CNG Dual Fuel Engine Knock Signal at 1400 rpm with Various Diesel to CNG Fuel Ratios”, Mendeley Data, V1, doi: 10.17632/jzmhw4mbn7.1

Figure: ./forecast_1440_720.png

Takeaway: During smooth engine working, Transformers with positional encoding has confidence to predict in next full cycle of four-stroke ICE (720 krank angles).
